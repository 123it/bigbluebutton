WELCOME

BigBlueButton web conferencing system provides voice (using Asterisk or FreeSWITCH) and video conference calling capabilitie as well as supports sharing slides, whiteboard and desktops. It's built using over fifteen open source components, runs on Mac, Unix, and PC computers, and is supported by a community that cares about good design and a streamlined user experience.

About BigBlueButton 0.72 release

The Bigbluebutton 0.72 release is the result of a joint effort between Coral CEA organization and Carleton University to support the development of the Bigbluebutton project. Bigbluebutton is an open source web conferencing system that provides solutions for for universities and businesses.
